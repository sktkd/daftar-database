Sign in with Google once to let the app add new items directly to your Sheet. Only needs the Sheets permission.
One-time Client ID setup:
1. console.cloud.google.com → Create project
2. APIs & Services → Enable Google Sheets API
3. Credentials → Create → OAuth 2.0 Client ID → Web application
4. Authorised JS origins: add your site URL (e.g. https://yourusername.github.io)
5. Copy Client ID → paste above → Sign in

How sync works
New entries appear in the app instantly. When saved, the row is copied to your clipboard — open your Google Sheet, go to the next empty row, and paste (Ctrl+V / ⌘V) to keep it in sync. To pull new rows from the Sheet into the app, tap ↻ Sync in the top bar.
